passport
function in module copernicus

Description

Valida si las credenciales del usuario son correctas.

Usage

new passport();

Properties

    Atributo utilizado para realizar consultas contra la base de datos. Es creado con el módulo 'seraph' utilizando la configuración de la base de datos contenida en 'dbConfig'.

  • dbConfig

    Contiene la dirección y credenciales de la base de datos.

  • localStrategy

    Módulo que proporciona una estrategia para la autenticación de usuarios utilizando nombre de usuario y contraseña.

    Módulo 'passport'.
