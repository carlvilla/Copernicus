notificacionesYCabeceraController
controller in module copernicus

Description

Este controlador está encargado de obtener las solicitudes del usuario para mostrarle notificaciones, y de obtener la información del usuario para mostrarla en la cabecera de la página.

Methods

    Muestra un mensaje de error.

    Respuesta de la API REST.

  • inicializacion()

    ParamTypeDetails
