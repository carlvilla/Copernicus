radioController
controller in module copernicus

Description

Este controlador es utilizado para comunicar al usuario con 'RadioManager', de modo que pueda enviar y recibir la URL de una emisora o canción.

Methods

  • cambiarEmisora()

    Indica a 'RadioManager' que el usuario quiere cambiar la URL utilizada.

  • inicializacion()

    Inicializa el controlador, obteniendo los datos de la sala y del usuario e inicializa 'RadioManager' enviándole el nombre de usuario del usuario y el id de la sala.

  • play()

    Indica a 'RadioManager' que el usuario quiere reproducir la emisora o canción.

  • setVolumen()

    Indica a 'RadioManager' que el usuario quiere cambiar el volumen.

Properties

  • sala

    Almacena los datos de la sala accedida.

  • usuario

    Almacena los datos del usuario.
