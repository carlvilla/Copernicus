PresentacionesManager
function in module copernicus

Description

Este manager se encarga de la gestión del servicio de presentaciones.

Usage

new PresentacionesManager();

Methods

  • actualizarPresentacion(mensaje)

    Actualiza la presentación que se muestra a partir de una enviada por otro usuario.

    Parameters
    ParamTypeDetails
    mensajeobject

    Mensaje con la presentación a mostrar.

  • cambiarPresentacion(presentacionParam)

    Cambia la presentación que se muestra y la envía a los usuarios conectados a la sala.

    Parameters
    ParamTypeDetails
    presentacionParamobject

    Mensaje con la presentación a mostrar.

  • inicializar(username, salaParam)

    Inicializa los valores de los atributoes 'iframe', 'usernameUsuario' y 'sala'. En el caso de que el atributo 'presentacion' tuviese algún valor, eso significa que otro usuario pasó una presentación antes de abrir el servicio por lo que se carga en el Iframe.

    Parameters
    ParamTypeDetails
    usernameString

    Nombre de usuario del usuario.

    salaParamString

    ID de la sala.

  • sendData(mensaje)

    Envía una presentación a los usuarios conectados a la sala a través del servidor de WebSockets.

    Parameters
    ParamTypeDetails
    mensajeobject

    Mensaje con la presentación.

Properties

  • iframe

    Iframe utilizado para mostrar las presentaciones.

  • presentacion

    Presentación que se muestra.

  • sala

    ID de la sala.

  • usernameUsuario

    Nombre de usuario del usuario.
