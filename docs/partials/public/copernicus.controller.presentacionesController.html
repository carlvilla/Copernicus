presentacionesController
controller in module copernicus

Description

Este controlador es utilizado para comunicar al usuario con 'PresentacionesManager', de modo que pueda cambiar la presentación y recibir las presentaciones de otros usuarios.

Methods

  • enviarFichero(file, errFiles)

    Envía la presentación seleccionada a 'PresentacionesManager'.

    Parameters
    ParamTypeDetails
    fileobject

    Contiene la presentación seleccionada por el usuario

    errFilesobject

    Indica que al obtener el fichero seleccionado por el usuario.

  • inicializacion()

    Inicializa el controlador, obteniendo los datos de la sala y del usuario e inicializa 'PresentacionesManager' enviándole el nombre de usuario del usuario y el id de la sala.

Properties

  • sala

    Almacena los datos de la sala accedida.

  • usuario

    Almacena los datos del usuario.
