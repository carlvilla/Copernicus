contactosController
controller in module copernicus

Description

Este controlador es utilizado para mostrar el listado de contactos, mostrar la información de los contactos y bloquearlos

Methods

  • bloquearContacto(username)

    Bloquea a un contacto seleccionado.

    Parameters
    ParamTypeDetails
    usernameString

    Nombre de usuario del contacto a bloquear.

  • error()

    Invocado si ocurrió un error al recuperar los contactos.

  • errorBloquear()

    Muestra una notificación indicando que hubo un error al bloquear al usuario.

  • filtrar()

    Filtra el listado de contactos.

  • inicializacion()

    Obtiene los contactos del usuario.

  • mostrarContacto(username)

    Muestra la información de un contacto seleccionado.

    Parameters
    ParamTypeDetails
    usernameString

    Nombre de usuario del usuario cuya información se quiere mostrar.

  • success()

    Almacena los contactos recuperados de la petición a la API REST.

  • successBloquear(res)

    Muestra una notificación indicando que el usuario fue bloqueado y actualiza la página.

    Parameters
    ParamTypeDetails
    resobject

    respuesta de la API REST

  • successMostrar(res)

    Obtiene de una respuesta de la API REST la información del contacto seleccionado.

    Parameters
    ParamTypeDetails
    resobject

    respuesta de la API REST

Properties

  • contactoSeleccionado

    Contacto seleccionado.

  • contactos

    Variable necesario para el filtrado de contactos.

  • textoFiltrar

    Texot para filtrar el listado de contactos.
