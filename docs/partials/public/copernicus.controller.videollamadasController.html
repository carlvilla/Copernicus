videollamadasController
controller in module copernicus

Description

Este controlador es utilizado para comunicar al usuario con 'VideollamadasManager', de modo que el usuario pueda participar a una videollamada y controlar el micrófono, el sonido y el video retransmitido a otros usuarios.

Methods

  • cerrarServicio()

    Indica a 'VideollamadasManager' que el usuario ha cerrado el servicio de videollamadas.

  • incializacion()

    Inicializa el controlador, obteniendo los datos de la sala y el usuario e inicializa 'VideollamadasManager' pasandole el nombre de usuario e ID de la sala.

  • setAltavoz()

    Indica a 'VideollamadasManager' que el usuario quiere silenciar o activar el altavoz.

  • setMicrofono()

    Indica a 'VideollamadasManager' que el usuario quiere silenciar o activar el micrófono.

  • stopVideo()

    Indica a 'VideollamadasManager' que el usuario quiere activar o desactivar el video.

Properties

  • muteAltavoz

    Booleano que indica si el altavoz está silenciado.

  • muteMicrophone

    Booleano que indica si el micrófono está silenciado.

  • sala

    Almacena los datos de la sala a la que se accedió.

  • usuario

    Almacena los datos del usuario.

  • video

    Booleano que indica si el video está desactivado.
