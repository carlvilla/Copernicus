registroController
controller in module copernicus

Description

Este controlador permite a un usuario registrarse.

Methods

  • comprobarCaracter(event)

    Método utilizado para que el usuario no pueda introducir espacios en blanco en el nombre de usuario.

    Parameters
    ParamTypeDetails
    eventobject

    Evento del cual se puede tener el código de la tecla introducida por el usuario.

  • error(err)

    Método llamado si el registro no se pudo completar. Muestra un mensaje uno u otro mensaje de error dependiendo si el problema fue del servidor o del usuario.

    Parameters
    ParamTypeDetails
    errobject

    Respuesta de la API REST.

  • fotoSeleccionada(evt)

    Obtiene la foto seleccionada por el usuario.

    Parameters
    ParamTypeDetails
    evtobject

    Contiene la foto seleccionada por el usuario.

  • registrar(usuario)

    Registra a un usuario en el sistema.

    Parameters
    ParamTypeDetails
    usuarioobject

    Datos del usuario que se quiere registrar.

  • success(res)

    Método llamado si el registro se completó con éxito. Crea una cookie en el navegador con el token de sesión y redirige al usuario a la página principal de la aplicación.

    Parameters
    ParamTypeDetails
    resobject

    Respuesta de la API REST.

Properties

  • foto

    Foto seleccionada por el usuario.

  • fotoPorDefecto

    Booleano que indica si se va a utilizar como foto de perfil la foto por defecto.

  • fotoRecortada

    Parte seleccionada por el usuario de la foto que subió.

  • messages

    Atributo utilizado para mostrar mensajes de error.

  • sizeMaxFoto

    Entero que indica el tamaño máximo permitido de las fotos.
