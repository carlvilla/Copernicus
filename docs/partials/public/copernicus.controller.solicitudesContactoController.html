solicitudesContactoController
controller in module copernicus

Description

Este controlador está encargado de gestionar las solicitudes de contacto, mostrandolas y permitiendo aceptarlas o ignorarlas.

Methods

  • aceptarSolicitud(usernameMandoSolicitud)

    Permite aceptar una solicitud de contacto.

    Parameters
    ParamTypeDetails
    usernameMandoSolicitudString

    Nombre de usuario del usuario que mando la solicitud de contacto.

  • cerrarPantallaSolicitudes()

    Cierra el dialogo de las solicitudes refrescando la página. De este modo los usuarios cuya solicitud de contacto fue aceptada aparecerán en el listado de contactos.

  • eliminarSolicitud(username)

    Elimina de la lista de solicitudes una solicitud que se acaba de aceptar o ignorar.

    Parameters
    ParamTypeDetails
    usernameString

    Nombre de usuario del usuario que mando la solicitud de contacto.

  • error(err)

    Muestra un mensaje de error debido a un problema con el servidor.

    Parameters
    ParamTypeDetails
    errobject

    Respuesta de la API REST.

  • ignorarSolicitud(usernameMandoSolicitud)

    Permite ignorar una solicitud de contacto.

    Parameters
    ParamTypeDetails
    usernameMandoSolicitudString

    Nombre de usuario del usuario que mando la solicitud de contacto.

  • inicializacion()

    Inicializa el controlador, obteniendo las solicitudes de contacto pendientes del usuario.

  • problemaAceptarSolicitud(res)

    Muestra una notificación si hubo un problema al aceptar la solicitud.

    Parameters
    ParamTypeDetails
    resobject

    Respuesta de la API REST.

  • solicitudAceptada(res)

    Muestra una notificación indicando que la solicitud fue aceptada

    Parameters
    ParamTypeDetails
    resobject

    Respuesta de la API REST.

  • solicitudIgnorada(res)

    Muestra una notificación indicando que la solicitud fue ignorada.

    Parameters
    ParamTypeDetails
    resobject

    Respuesta de la API REST.

Properties

  • solicitudes

    Listado de solicitudes de contacto pendientes.
