chatTextoController
controller in module copernicus

Description

Este controlador es utilizado para comunicar al usuario con 'ChatTextoManager', de modo que pueda enviar y recibir mensajes y archivos.

Methods

  • enviarFichero(file, errFiles)

    Envía un fichero.

    Parameters
    ParamTypeDetails
    fileobject

    Fichero que el usuario envía

    errFilesobject

    Error al recibir el fichero.

  • enviarMensaje(mensaje)

    Envía un mensaje del usuario.

    Parameters
    ParamTypeDetails
    mensajeString

    Mensaje enviado por el usuario.

  • enviarMensajeTeclado(event, mensaje)

    Envía un mensaje del usuario al pulsar Enter en el teclado.

    Parameters
    ParamTypeDetails
    eventobject

    Evento ocasionado al pulsar una tecla.

    mensajeString

    Mensaje enviado por el usuario.

  • inicializacion()

    Inicializa el controlador. Se encarga de inicializar el 'ChatTextoManager' pasandole el nombre de usuario y la sala, y recibe los mensajes enviados.

Properties

  • mensajes

    Mensajes del chat de texto.

  • sala

    Id de la sala a la que se accedió.

  • usuario

    Usuario que está utilizando la sala.
