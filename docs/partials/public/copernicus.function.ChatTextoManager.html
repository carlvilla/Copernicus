ChatTextoManager
function in module copernicus

Description

Este manager se encarga de la gestión del chat de texto.

Usage

new ChatTextoManager();

Methods

  • addMensaje(mensaje)

    Guarda un mensaje recibido de modo que se mostrará en el chat de texto, y mueve el scroll del chat de texto hasta el último mensaje.

    Parameters
    ParamTypeDetails
    mensajeString

    Mensaje a añadir.

  • enviarMensaje(mensaje)

    Envía un mensaje a través de 'sendData' al resto de usuarios conectados a la sala, y lo muestra en el chat de texto.

    Parameters
    ParamTypeDetails
    mensajeString

    Mensaje a enviar al resto de usuarios.

  • getFeedback(feedback)

    Obtiene feedback de un mensaje enviado, de modo que el usuario puede saber el número de usuarios que recibieron el mensaje.

    Parameters
    ParamTypeDetails
